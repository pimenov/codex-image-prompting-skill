# Notices and attribution

This package is an independent prompt-only Codex skill inspired by image prompt
craft patterns from:

- Repository: `wuyoscar/GPT-Image2-Skill`
- URL: https://github.com/wuyoscar/GPT-Image2-Skill
- License: MIT

The upstream project includes a broader workflow around GPT Image 2, reference
galleries, CLI execution, and local API use. This package is not a mirror. It
keeps only the reusable prompt-design ideas that are useful for Codex Desktop
image generation and intentionally excludes local API wrappers, API-key setup,
paid generation code paths, and packaged CLI execution.

The public examples in this package are generic and do not include private
brands, private project paths, private tracker links, or private generated
assets.
