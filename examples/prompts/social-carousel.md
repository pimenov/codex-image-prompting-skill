# Example: social carousel

User request:

```text
Generate a five-slide Threads carousel about turning task chaos into a clear plan. Use short readable text inside each image.
```

Prompt plan:

```text
Carousel mode:
separate-slide carousel

Visual system for all slides:
Use the user's requested style if provided. If not, use a clean, modern, high-contrast visual system that fits the topic. Keep the same palette, lighting, character/object treatment, and background grammar across all slides. Do not default to a recurring house palette from prior examples.

Slide 1:
Square 1:1 Threads carousel slide 1 of 5.
Render the exact large headline: "TOO MANY THREADS?"
Scene: scattered notes, messages, and question marks orbit a stressed but calm operator.

Slide 2:
Square 1:1 Threads carousel slide 2 of 5.
Render the exact large headline: "FIND THE SIGNAL"
Scene: duplicated tasks and unclear ownership collapse into a small set of highlighted signals.

Slide 3:
Square 1:1 Threads carousel slide 3 of 5.
Render the exact large headline: "MAKE WORK VISIBLE"
Scene: a structured planning board appears with abstract task cards and ownership markers.

Slide 4:
Square 1:1 Threads carousel slide 4 of 5.
Render the exact large headline: "RUN IN SLICES"
Scene: people and agents collaborate around a shared workflow, with clear sequential steps.

Slide 5:
Square 1:1 Threads carousel slide 5 of 5.
Render the exact large headline: "SHIP WITH EVIDENCE"
Scene: a clear roadmap, checklist, and glowing proof trail resolve the visual story.
```

Expected review note:

```text
Check whether every slide has one readable message, the same visual system, and no random extra text. If the text fails, simplify the headline and regenerate that slide.
```
