# Example: seamless carousel

User request:

```text
Prepare a prompt for a five-slide seamless carousel about turning task chaos into a clear plan.
```

Polished prompt:

```text
Carousel mode:
concept

Master prompt:
Create one ultra-wide 5:1 panoramic concept artwork for a seamless social media carousel, designed to suggest five square slides.

Theme:
"From task chaos to a clear plan"

Slide plan:
1. Hook scene - scattered notes, messages, and question marks; blank message zone for "Too many threads?"
2. Problem scene - duplicated tasks and unclear ownership; blank message zone for "Find the signal"
3. Method scene - a structured planning board appears; blank message zone for "Make the work visible"
4. Execution scene - people and agents collaborate around a shared workflow; blank message zone for "Run in slices"
5. Result scene - a clear roadmap and checklist; blank message zone for "Ship with evidence"

Continuity:
A single glowing blue line travels across all five slides. It starts as scattered sparks, becomes a path, turns into a structured graph, connects collaborators, and resolves into an arrow.

Visual system:
Editorial systems-thinking illustration, warm off-white background, graphite linework, blue and amber accents, paper texture, clean geometric composition.

Output constraints:
Continuous world, no hard panel borders, no random words, no dense microtext. This is concept art, not final production geometry.

Slice notes:
For production, build a deterministic 5400x1080 master canvas for five 1080x1080 slides, overlay exact typography after generation, and keep important text inside safe zones.
```
